Bookmanize - Browser extension

Advanced bookmarks manager

Bookmarks manager with a clean user interface where all bookmarks can be easily accessed in one full list view with tagging, instant sorting, and filtering. Organize your bookmarks with custom tags and faster navigates between them.

About

We are an independent team of developers who love what they do:)

Features

All bookmarks on one screen
Fast access to all main functions
Instant searching, filtering, sorting, adding, deleting bookmarks
Drag n drop bookmarks and folders

All the features that the default bookmarks have in a more user-friendly interface with high performance for a large number of bookmarks
